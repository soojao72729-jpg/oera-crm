# OERA CRM Mobile - Istemal Karne ki Hidayat (Guide)

Ye software ab mobile par aik asli App ki tarah chal sakta hai. Isay istemal karne ka process neeche diya gaya hai:

---

## 1. Mobile Par Install Karne ka Tareeqa
Agar aap chahte hain ke worker ke mobile par ye software aik icon ki tarah nazar aaye:
1. Is poore folder ko **Google Drive** or **GitHub** par upload karein.
2. Mobile ke **Chrome Browser** mein uska link kholen.
3. Chrome ke menu (uper 3 dots) par click karein aur **"Install App"** ya **"Add to Home Screen"** par click karein.
4. Ab aapke mobile menu mein OERA CRM ka icon aa jayega.

---

## 2. Worker ke liye (Kaam aur Sharing)
Worker ko ye kaam karna hoga:
1. Mobile par App kholen (Internet ki zaroorat nahi hai, ye offline kaam karega).
2. Leads add karein ya call logs save karein.
3. Kaam khatam hone par side mein **Cloud (Sync Hub)** icon par click karein.
4. **"Share Backup via WhatsApp"** wala hara (green) button dabayen.
5. WhatsApp par apne Admin (Aap) ko select karein aur file bhej dein.

---

## 3. Admin ke liye (Data Sync karna)
Jab worker file bhej de, toh aap ye karein:
1. WhatsApp se file download karein.
2. Apne mobile/computer par OERA App kholen.
3. **Sync Hub (Cloud Icon)** par jayen.
4. **"Import Database"** ka button dabayen.
5. Wo file select karein jo worker ne bheji thi.
6. **Bas!** Worker ka sara kaam aapke dashboard par update ho jayega.

---

## Zaroori Note:
- Agar browser mangay, toh isay "Edit/Save" ki ijazat (Permission) dein.
- Mobile par hamesha Chrome ya Safari browser ka istemal karein.
- Har worker ka apna alag password aap **Database (Admin Panel)** se bana kar de sakte hain.
